# Dynamic Appointment Booking System

## 📌 Overview

The Dynamic Appointment Booking System is a web-based application developed to simplify appointment scheduling between users and administrators. The system provides a user-friendly interface for booking appointments while allowing administrators to manage bookings efficiently.

---

## 🚀 Features

- User Registration & Login
- Admin Login
- Appointment Booking
- Appointment History
- Dashboard
- Appointment Status Update
- Responsive User Interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
